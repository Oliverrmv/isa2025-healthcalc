# isa2025-healthcalc
Health calculator used in Ingeniería del Software Avanzada

En este proyecto desarrollaremos una calculadora que nos permita visualizar parámetros en realción al peso ideal de una persona o su TMB (Tasa metabólica basal)

para la correcta implementación de los métodos usados para la obtención de parametros hemos hecho uso de de los test proporcionados por JUnit5
